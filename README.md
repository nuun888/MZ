### バグ、競合報告について
バグ、競合報告は基本的にツクールフォーラム(日本)のスレッド、会話、X(Twitter)のDM、githubでお願いいたします。  
Xでの返信の場合はスーパーゲー制デー、プレミアムツクールデー前日から当日を避けての返信をお願いいたします。対応に時間がかかる場合は上記の場所で再投稿をお願いする場合がありますのでご了承ください。  
githubでの報告は、報告の確認までに時間がかかる場合があります。  
要望はツクールフォーラム(日本)のスレッド、会話、X(Twitter)のDMのみ対応いたします。  

原則、最新バージョン前提で対応しますので旧バージョンをご使用の場合は必ず使用中のバージョンを記載してください。最悪ゲームが動かなくなる恐れがあります。  
バグ、競合対応が優先になります。バグ、競合対応が立て込んでいる場合は要望の対応が遅れる場合がありますのでご了承ください。  
大がかりな変更が必要な要望に関しては、対応可能か判断までに時間が掛かる場合がございます。バグ、競合、要望対応が立て込んでいる場合は、返信までに１か月以上かかる場合があります。  
バトルスタイル拡張、メニュー画面の旧版の要望の対応は行いません。  
難読化を行っているプラグインとの競合対応、改変したプラグイン、コアスクリプトVer.1.6.0以前のバージョンのサポートは行いません。  

バグ、競合、要望の対応順は変更になる場合がございます。  
有償プラグイン、支援者の依頼が優先になります。  

## 要望に関して
要望は基本的にバグ、競合対応がある程度処理を終えた時に順次対応いたします。  
要望受付停止期間中に要望を行った場合は、支援者以外の方は原則お断りさせていただきます。  
多忙時期、長期更新停止、依頼が混雑している時期は全ての対象者で要望をお断りさせていただきます。  

要望は、可能か判断してから返信を行うため、数週間程度返信がない場合がありますのでご了承ください。  
返信がないと言われましても順番の関係上対応できません。  
原則、プラグインのコンセプトから一脱するような内容、システムの大がかりな改修が必要になる内容の要望は対応を行いません。  

[既知の不具合、更新予定](https://github.com/nuun888/MZ/blob/master/bug.md)  

[よくある質問](https://github.com/nuun888/MZ/blob/master/README/q&a.md)  

Ver.1.6以降の対応のみにさせていただきます。  
それ以前のバージョンでは正常に動作しない場合がございます。
