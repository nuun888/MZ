# Ver.1.0.6  
 [ダウンロード](https://raw.githubusercontent.com/nuun888/MZ/master/NUUN_BattlerOverlayBase.js)  
 
モンスター、サイドビューアクター上に画像を表示させるベースプラグイン。  
戦闘中にモンスター画像の色相を変更したり、攻撃時にモンスター画像と一緒にフラッシュをしないようにします。
