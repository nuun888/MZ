# Ver.1.6.0

#### 必須、前提プラグイン
[サポートアクター](https://github.com/nuun888/MZ/blob/master/README/SupportActor.md)  

参加しているサポートアクターを表示させます。  
このプラグインはサポートアクター（NUUN_SupportActor）Ver.2.0.0以降の拡張機能です。
