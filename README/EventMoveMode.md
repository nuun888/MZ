イベントの移動を乗り物と同じ移動方式に変更します。  

## 設定
イベントのメモ欄及び、イベントの実行内容の注釈で記述  
`<EventMoveMode:[id]>` 上記のIDモードでの通行になります。イベントのメモ欄では全てのページで有効になります。  
イベントの実行内容に記入の場合はそのページでのみ有効になります。  
`[id]`:下記のID  
1:小型船と同じ  
2:大型船と同じ  
3:飛行艇と同じ  

注釈での記入。
