## 設定方法
### 最大所持金額
所持金の最大金額を設定します。-1で無制限になります。0でデフォルト上限値　1以上で設定できる範囲の上限値  
### カンマ区切り
所持金をカンマ区切りにします。
### アイコンインデックス
所持金を表示するときに左側にアイコンを表示させます。表示させるウィンドウをアイコン表示クラスで指定できます。
