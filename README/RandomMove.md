移動による方向キーの入力をランダム化します。  

当プラグインではマウス操作での移動を無効にする機能はありません。  
マウスでの操作では方向キーのランダム化されません。  
マウス操作での移動を無効にする場合は別途プラグインが必要です。  

## 設定
### 特徴での方向キーのランダム化
特徴を有するメモ欄  
`<RomdomKeyMove>` この特徴があるアクターがパーティ内に存在する場合、方向キーがランダムになります。  

### スイッチでの方向キーのランダム化
プラグインパラメータ「ランダム方向スイッチ」での設定を行っている場合、スイッチがONの場合、方向キーがランダムになります。  
特徴でランダム操作になっている場合は方向キーの再配置は行われません。  

### リージョンでのキー操作の指定
特定のリージョン上で移動を行う際に、プラグインパラメータ「リージョンランダム方向」で対象のリージョンでのキー指定が設定されている場合、
対象のリージョンID上の移動の方向キーは、設定されたキーの割り当てになります。ランダムには対応しておりません。  

### 特定のマップでは方向キーの指定、ランダム化を無効にする。
プラグインパラメータ「無効マップID」で無効にするマップIDを指定してください。複数指定できます。
