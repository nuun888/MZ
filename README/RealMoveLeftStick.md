#### 必須、前提プラグイン
[キーボタン割り当て](https://github.com/nuun888/MZ/blob/master/README/UserKey.md)Ver.1.2.2以降  

ゲームパッドの左スティックの倒した強さによりプレイヤーの移動速度を変化させるように変更します。  
このプラグインはキーボタン割り当てプラグイン(Ver.1.2.1以降)が必要となります。
