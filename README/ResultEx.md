# リザルト
## Ver.1.0.0
[販売先](https://booth.pm/ja/items/8046406)  

戦闘終了時にリザルト画面を表示します。  

## 設定
アイテム、武器、防具のメモ欄  
`<NoResultDropList>` アイテム入手のドロップリストに表示しません。  

スキルのメモ欄  
`<NoResultLearnSkillList>` 習得スキルのスキルリストに表示しません。  

#### 勝利後リザルト画面遅延フレーム数
勝利後リザルト画面が表示されるまでのフレーム数を指定します。0は戦闘終了時に瞬時に表示します。  

#### キャンセルリザルト終了
キャンセルキーを押したときにリザルトを終了します。OFFの場合はキャンセルキーで決定キーと同じ動作をします。  

#### リザルトウィンドウモード(バトルのみ)
Window:通常のウィンドウと同じく、ウィンドウが重なるとマスク処理がかかります。アクター画像はウィンドウの正面に表示されます。  
Normal:ウィンドウのマスク処理は行われません。背景画像を設定している場合は強制的にこのモードになります。   
アクター画像、背景画像はウィンドウ背後に表示されます。  

#### アクターの画像
勝利時のアクター、レベルアップ時にアクターの画像を表示させることが出来ます。  
アクターの表示に動きを付ける場合は、プラグインパラメータのアクター演出で設定します。  
アクター演出の一番上は初期状態の表示位置を指定します。フレーム数は無視されます。  
設定方法はベントコマンドのピクチャと同じです。  

#### APNG
勝利時のアクター、レベルアップ時に表示するアクターにAPNGで表示できます。  
表示するにはトリアコンタン様のApngPictureプラグイン及び以下のライブラリが必要です。  
github.com/sbfkcel/pixi-apngAndGif  
github.com/sbfkcel/pixi-apngAndGif/blob/master/dist/PixiApngAndGif.js  
ウィンドウ内に表示されるアクター画像はAPNG対象外になります。  

#### 評価式パラメータ（全共通）
各項目の評価式のパラメータ  
actor:アクターのゲームデータ  
dactor:アクターのデータベースデータ  
v:ゲーム変数 v[2]でゲーム変数2番の値  

#### 操作
決定キー:ページ切り替え(レベルアップアクターがいなければリザルト終了)  
キャンセル:ページ切り替え(キャンセルリザルト終了をONに設定している場合はリザルト終了)  
QW:ドロップ、習得スキルページ切替  

### 勝利時ボイス
勝利時にランダムに一人ボイスが再生されます。  
レベルアップ時にボイスが再生されます。  
ボイスが複数設定されている場合は、ランダムに再生されます。  
ボイスの設定がされていないアクターは再生対象にされません。  
アクター毎に専用のME、BGMを再生できます。  

### リザルトアクター
勝利後に表示されるリザルトにアクターを表示します。  

#### 対象モード
最後に攻撃したアクター:アクターは最後に攻撃したアクターの画像が表示されます。  
与えた合計ダメージの大きいアクター:与ダメージが一番高いアクターの画像が表示されます。  

### リザルトエフェクト
戦闘終了時に任意のエフェクトを表示させます。  
エフェクト再生中はキー操作は実行できません。  

### リザルト時ウィンドウスプライト非表示
リザルト表示時に指定のウィンドウ、スプライトを非表示にします。
